PRIVACY POLICY

개인정보 수집·이용 안내

WHITE MARKET은 시큐어코딩 과제용 중고거래 플랫폼으로, 회원 식별과 거래 기능 제공에 필요한 범위에서 개인정보를 처리합니다.

1. 수집 항목

  • 필수: 로그인 아이디, 이름, 닉네임, 비밀번호 해시, 휴대전화 번호, 시·도, 시·군·구, 읍·면·동, 상세 주소
  • 자동 생성: 가입일, 동의 시각, 로그인 세션 정보, 신고·송금·서비스 이용 기록
  • 선택: 공개 소개글

2. 이용 목적

  • 회원가입, 로그인, 계정 식별과 중복 아이디·닉네임 방지
  • 중고거래 연락과 거래 지역 관리
  • 악성 사용자·상품 신고 처리와 부정 이용 방지
  • 보안 사고 조사와 관리자 접근 통제

3. 주소 입력 방식

외부 주소 검색 API를 사용하지 않습니다. 사용자가 시·도, 시·군·구, 읍·면·동, 번지·건물명·동·호수 등의 상세 주소를 직접 입력하며 서버에서 각 항목의 길이와 제어 문자를 검증합니다.

4. 공개 범위

일반 공개 프로필에는 닉네임, 로그인 아이디, 소개글, 가입일과 판매 상품만 표시합니다. 이름, 전화번호, 주소, 잔액, 동의 기록과 신고 횟수는 일반 사용자에게 공개하지 않습니다.

5. 안전성 확보 조치

  • 비밀번호 단방향 해시 저장
  • CSRF 토큰, 세션 쿠키 보호, 로그인 반복 시도 제한
  • SQL 인자 바인딩과 템플릿 자동 이스케이프
  • 관리자 권한과 상품·게시물 소유권 검사
  • 민감 페이지 캐시 방지와 보안 응답 헤더 적용

6. 이용자의 권리

사용자는 마이페이지에서 소개글, 연락처와 주소를 수정하고 별도 페이지에서 비밀번호를 변경할 수 있습니다.

시행일: 2026년 7월 23일
